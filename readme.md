# Sketch Utils

Helper functions for creating Sketch plugins.

## Usage

+ Git clone or subtree

```bash
git subtree add --prefix vendor https://github.com/dunckr/sketch-utils master --squash
```

+ Import into sketchplugin file

```javascript
@import 'vendor/index.js'
```
